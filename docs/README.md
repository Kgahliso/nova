# Nova Energy Invest Website

A modern, responsive website for Nova Energy Invest - Solar & Green Renewable Energy solutions.

## Project Structure

```
novaenergy/
├── index.html              # Homepage
├── 404.html               # Error page
├── pages/                 # Main content pages
├── services/              # Service pages
├── projects/              # Project showcase pages
├── blog/                  # Blog and articles
├── pricing/               # Pricing information
├── assets/                # Static assets
│   ├── css/              # Stylesheets
│   ├── js/               # JavaScript files
│   ├── images/           # Images
│   ├── videos/           # Video content
│   └── favicon.ico       # Site icon
├── includes/              # Reusable components
├── api/                   # Backend/API files
└── docs/                  # Documentation
```

## Features

- Responsive design
- Solar energy services showcase
- Project portfolio
- Blog for industry insights
- Contact forms
- Testimonials
- FAQ section
- Pricing plans

## Technologies

- HTML5

- Fancybox

## Setup

1. Clone or download the repository
2. Open `index.html` in a web browser
3. For development with a local server:
   ```bash
   npm install
   npm start
   ```

## File Organization

- **Pages**: All main content pages in `/pages/`
- **Services**: Service-related content in `/services/`
- **Projects**: Project showcases in `/projects/`
- **Blog**: Blog posts and articles in `/blog/`
- **Assets**: All static files (CSS, JS, images) in `/assets/`
